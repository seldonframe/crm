# Demo GIF Runbook (Step 18)

Goal: 30-second flow showing framework differentiation.

## Storyboard

1. Fork repository
2. Run app (`pnpm dev`)
3. Sign up + Soul setup wizard
4. Show labels/pipeline changing for niche
5. Open dashboard + contacts + deals

## Capture

Recommended tools:
- ScreenToGif (Windows)
- Kap (macOS)
- OBS + ffmpeg conversion

Suggested settings:
- 1440x900 or 1280x720 capture area
- 15 fps for GIF export
- keep under 10MB for README

## Export

- Output file: `public/demo/soul-setup.gif`
- Optional poster: `public/demo/hero.png`
